⚡️

Blazing fast

Built with performance in mind, this package is blazing fast and provides a performant tool for your file management needs.

📦️

Nova 4 compatible

Made specifically for Laravel Nova 4. This is a plug'n'play package you can install and start using right away !

💅

Built with a robust stack

Built with Tailwindcss 3, Vue 3 and Vuex. This stack provides a robust base for the package and insures future update to go as smoothly as possible.

💽

Multi disk support

With multi disk support, you can switch between defined filesystem disks and start using them in your Nova application.

🧩

Chunk uploads

Chunk uploads provide a performant way to handle uploads when dealing with large files.

Various customization and configuration options
